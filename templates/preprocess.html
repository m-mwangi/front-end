Understanding the Dataset

The dataset used for maternal health risk prediction was sourced from Kaggle and comprises 1,014 records with 6 key health indicators and a target classification.

The features included in this dataset are: Blood Pressure (SystolicBp, DiastolicBP), Body Temperature, Blood Sugar, Heart Rate, and Age. The output variable, Risk Level, categorizes maternal health risks into three distinct groups:

  • High Risk: Requires urgent medical attention.
  • Mid Risk: Needs monitoring and lifestyle adjustments.
  • Low Risk: Indicates stable maternal health.

This project aims to leverage machine learning techniques to predict maternal risk levels accurately, thus enabling real-time interventions and improved maternal healthcare outcomes.

Data Pre-processing Steps

1. Data Loading

- The dataset is loaded into a Pandas dataframe.

- This allows easy manipulation of the data.

2. Inspecting Data

- We analyze distributions, missing values, and structure using statistical summaries and visualizations.

3. Encoding Categorical Variables

- The categorical variable (Risk Level) is converted into numerical values.

4. Checking for Outliers

- Outliers are identified using boxplots.

- They are capped to prevent skewing model performance.

5. Scaling the Data

- Standard Scaling is applied to normalize feature ranges for uniformity.

6. Balancing the Dataset

- SMOTE generates synthetic samples for the underrepresented classes.

7. Splitting the Data

- The dataset is split into training, validation, and testing sets.

8. Dive into Building the Model

- The preprocessing process is complete.
